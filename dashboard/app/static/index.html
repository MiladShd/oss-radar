Which packages are gaining momentum — and which are becoming risky dependencies?

A self-managing pipeline ingests downloads, GitHub activity, dependency graphs, vulnerabilities and security-health metrics every day, then trains gradient-boosted models to forecast adoption momentum and dependency risk across the Python / AI ecosystem. AI agents retrain the models, write the brief, and open a PR.
